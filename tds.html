Technical Data Sheet

Description
Fully synthetic 2-stroke engine oil for two-stroke engines formulated from synthetic base fluids and advanced performance additives.

Application
Suitable for oil-fuel mix (pre-mix) and oil injection lubrication systems - follow manufacturers recommendations for oil-fuel ratio.

Features
Its high lubricity helps protect the engine from piston scuffing and component wear. It also helps control engine deposits and exhaust system blocking, while extending engine life without sacrificing power. The oil contains a synthetic base fluid that reduces visible exhaust smoke, and low ash additives that prevent pre-ignition. Additionally, the oil is easily miscible with fuel, which ensures a stable and homogeneous mixture even in cold temperatures.

Physical Characteristics

Property Value
Viscosity @40 ºC ASTM D 445, CST: 37.5
Viscosity @100 ºC ASTM D 445, CST: 7.5
Viscosity Index ASTM D 2270: 170
Flash Point, COC ASTM D 92: 92
Pour Point, ºC ASTM D97: -38
Density @ 15 ºC ASTM D 4052: 0.868
TBN (mg KOH/gm) ASTM D 2896: 1.6

Performance Levels
API TC, JASO FD
